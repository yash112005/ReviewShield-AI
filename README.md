# 🛡️ ReviewShield AI

**ReviewShield AI** is an AI-powered Fake Review Detection system that uses **Natural Language Processing (NLP)** and **Machine Learning** to identify whether a customer review is genuine or fake. The project helps improve online review authenticity, protect consumers from deceptive feedback, and assist businesses in maintaining trustworthy platforms.

---

## 🚀 Features

- 🔍 Detects fake and genuine customer reviews
- 🤖 Machine Learning-based classification
- 📝 Natural Language Processing (NLP) preprocessing
- 📊 Review sentiment and authenticity analysis
- ⚡ Fast prediction for new reviews
- 📈 Trained on both a small demo dataset and a large-scale Kaggle dataset (40,000+ reviews)

## 📊 Dataset

This project uses two datasets:

### 📁 Demo Dataset
- A **200-review dataset** is included for demonstration and development purposes.
- The notebook available in this repository is trained using this dataset.

### 📁 Full Training Dataset
- A **40,000+ review Kaggle dataset** was used to train and evaluate the complete model in Google Colab.
- The dataset is **not included** in this repository due to licensing and size limitations.
